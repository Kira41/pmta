# Shiva Env Variables — نسخة مُحسنة (Lifecycle + Functions + Code Quotes)

هذا الإصدار يعالج الملاحظات السابقة مباشرةً:
- **Grouping واضح** حسب مراحل التشغيل.
- **Cycle life** لكل متغير (من لحظة القراءة وحتى نقطة التأثير التشغيلي).
- **Function-level mapping**: أسماء الدوال التي تتأثر بكل متغير.
- **Code quote** فعلي من `shiva.py` لكل متغير.
- **سيناريوهات أدق** (على الأقل سيناريوهان لكل متغير).

> ملاحظة: هذا التقرير يغطي فقط المتغيرات المقروءة مباشرة عبر `os.getenv("...")` داخل `shiva.py` (عددها 60).

## A) خارطة دورة الحياة العامة (Global lifecycle)
1. **Startup Parse**: عند تحميل `shiva.py`، يتم قراءة env وتطبيع الأنواع (`int/float/bool/str`) مع fallbacks آمنة.
2. **Runtime Decision**: الدوال التشغيلية تستخدم القيم أثناء التحقق، health checks، backoff، وbridge polling.
3. **Background Loops**: بعض القيم تضبط حلقات دورية (PMTA/Bridge polling) وتأثيرها مستمر أثناء عمل الخدمة.
4. **Job Execution**: أثناء `smtp_send_job*` تؤثر القيم على pacing, filtering, diagnostics, and gating.
